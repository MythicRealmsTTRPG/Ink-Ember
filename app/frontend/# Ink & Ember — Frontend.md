# Ink & Ember — Frontend

This directory contains the frontend application for **Ink & Ember**, a worldbuilding and narrative management platform for writers and tabletop creators.

The frontend is responsible for:
- Rendering the user interface
- Managing themes and presentation state
- Communicating with the backend API
- Providing a fast, readable, wiki-style experience for large, long-running creative projects

## Structure

frontend/
├── src/
│ ├── contexts/ # React context providers (themes, global state, etc.)
│ ├── Index.css # Global styles
│ └── main entry files (WIP)
├── README.md # This file
└── package.json # Frontend configuration (to be added)


## Tech Direction (Planned)

- **Framework:** React
- **Build Tool:** Vite
- **Styling:** CSS / Tailwind (TBD)
- **State Management:** React Context (with potential expansion)
- **API Layer:** REST / JSON (served by the backend)

## Status

🚧 Early foundation stage  
This frontend is under active development and will evolve as Ink & Ember’s data model and backend stabilize.
